# Catch the Bus – Jump and Run Game

Ein 2D-Jump-and-Run-Spiel, entwickelt mit JavaScript und dem Phaser.js Framework.

## Über das Projekt

Dieses Projekt entstand im Rahmen des Hochschulmoduls „Rich-Media-Anwendungen“. Ziel war die Entwicklung eines interaktiven Browsergames unter Einsatz moderner Webtechnologien und grundlegender Konzepte der Spieleentwicklung.

Im Spiel verpasst die Spielfigur den Bus und muss zur Schule sprinten. Dabei müssen Hindernisse überwunden und Booster eingesammelt werden.

## Tech Stack

| Bereich            | Technologien |
| ------------------ | ------------ |
| Game Framework     | Phaser.js    |
| Programmiersprache | JavaScript   |
| Webtechnologien    | HTML, CSS    |

## Features

* 2D-Jump-and-Run-Gameplay
* Spielfigurauswahl
* Steuerung der Spielfigur
* Hindernisse und Kollisionserkennung
* Sammelobjekte (Booster)
* Level- und Bewegungslogik
* Integration von Hintergrundmusik und Soundeffekten
* Interaktive Spielmechaniken

## Lernziele

* Einstieg in die Spieleentwicklung mit Phaser.js
* Umsetzung von Spiellogik mit JavaScript
* Arbeit mit Kollisionserkennung und Animationen
* Strukturierung eines interaktiven Browsergames
